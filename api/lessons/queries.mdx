## lessons

Retrieves a paginated list of lessons with optional filtering.

**Source**: `LessonResource.kt:57`

### Parameters

<ParamField path="pagination" type="PaginationParams" default="{}">
  Pagination settings with `page` and `size` fields
</ParamField>

<ParamField path="lessonStatus" type="LessonStatus">
  Filter by lesson status: `PENDING`, `CONFIRMED`, `CANCELED`, `REJECTED`, `COMPLETED`
</ParamField>

<ParamField path="lessonType" type="ClassroomType">
  Filter by classroom type: `INDIVIDUAL`, `GROUP`, `HOME`
</ParamField>

<ParamField path="paymentMode" type="PaymentMode">
  Filter by payment mode: `FREE`, `PAID`, `SINGLE`
</ParamField>

<ParamField path="startDate" type="String">
  Filter lessons starting from this date (ISO format)
</ParamField>

<ParamField path="endDate" type="String">
  Filter lessons up to this date (ISO format)
</ParamField>

<ParamField path="search" type="String">
  Search term for filtering lessons
</ParamField>

### Returns

`List<Lesson>` - Array of lesson objects

### Example

```graphql
query GetLessons {
  lessons(
    pagination: { page: 0, size: 20 }
    lessonStatus: PENDING
    paymentMode: PAID
    startDate: "2026-03-01"
    endDate: "2026-03-31"
  ) {
    id
    uuid
    lessonDate
    lessonTime
    status
    minutes
    subject {
      name
    }
    classroom {
      signature
    }
  }
}
```

## checkLessonTime

Validates if a lesson time slot is available for a given classroom.

**Source**: `LessonResource.kt:38`

### Parameters

<ParamField path="lessonDate" type="String" required>
  The lesson date in ISO format (YYYY-MM-DD)
</ParamField>

<ParamField path="lessonTime" type="String" required>
  The lesson start time in ISO format (HH:mm:ss)
</ParamField>

<ParamField path="classroomSignature" type="String" required>
  The unique signature of the classroom
</ParamField>

<ParamField path="minutes" type="Long" default="0">
  Duration of the lesson in minutes
</ParamField>

<ParamField path="currentLessonId" type="BigInteger">
  ID of current lesson (used when updating existing lesson)
</ParamField>

### Returns

`List<LocalDateTime>` - Array of conflicting time slots, empty if available

### Example

```graphql
query CheckAvailability {
  checkLessonTime(
    lessonDate: "2026-03-15"
    lessonTime: "14:00:00"
    classroomSignature: "classroom-sig-123"
    minutes: 60
  )
}
```

## initBookLesson

Initializes the lesson booking process and retrieves all necessary information for booking.

**Source**: `BookLessonResource.kt:31`

### Parameters

<ParamField path="data" type="InitBookLessonDto" required>
  Initialization data containing:
  - `lessonResponseUUID` - UUID from lesson request response
  - `classRoomSig` - Classroom signature
  - `lessonUuid` - UUID of existing lesson (optional)
</ParamField>

### Returns

`InitBookLessonOutDto` - Complete booking initialization data including:
- Lesson type and rates
- Available vouchers and packets
- Student and tutor information
- Payment transaction details

### Example

```graphql
query InitializeBooking {
  initBookLesson(
    data: {
      lessonResponseUUID: "123e4567-e89b-12d3-a456-426614174000"
      classRoomSig: "classroom-sig-123"
    }
  ) {
    lessonType
    freeLesson
    hasVouchers
    voucherHours
    hasPackets
    packetHours
    lessonRate {
      id
      hourlyRate
      currency
    }
    tutor {
      id
      firstName
      lastName
    }
    student {
      id
      firstName
      lastName
    }
    isFirstBookLesson
    isHomeLesson
  }
}
```
