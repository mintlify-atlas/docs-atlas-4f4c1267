## Single Classroom

### singleClass

Retrieve a single classroom by its ID, including user-specific information.

<ParamField query="classroomId" type="BigInteger" required>
  The unique identifier of the classroom
</ParamField>

<ResponseField name="classroom" type="MyClassroom">
  Returns the classroom with user-specific context, or null if not found
</ResponseField>

```kotlin
// ClassroomResource.kt:31
@Query
fun singleClass(classroomId: BigInteger): MyClassroom? {
    val classroom = classroomService.getSingleClassroom(classroomId) 
        ?: throw Exception("Classroom not found")
    return classroomService.getMyClassesWithClassroomId(classroom.id)
}
```

## Student Classrooms

### studentClasses

Retrieve paginated list of classrooms for the authenticated student (group classes only).

<ParamField query="pagination" type="PaginationParams" default="{}">
  Pagination parameters with page and size
  - `page`: Page number (default: 0)
  - `size`: Items per page (default: 10)
</ParamField>

<ResponseField name="classrooms" type="List<MyClassroom>">
  Returns list of classrooms visible on cluster
</ResponseField>

```kotlin
// ClassroomResource.kt:45
@Query
fun studentClasses(
    @Name("pagination") @DefaultValue("{}") paginationParams: PaginationParams
) = identityUtil.authImpersonateAccount {
    val classroomIds = classroomService.getClassroomsPaged(
        studentId = it.id,
        classRoomType = ClassroomType.MULTIPLE,
        page = paginationParams.page!!,
        size = paginationParams.size!!
    ).filter { classroom -> classroom.isVisibleOnCluster }
     .map { classroom -> classroom.id }
    
    classroomService.getMyClassesWithClassroomIds(classroomIds = classroomIds)
}
```

### userClasses

Retrieve classrooms for the authenticated user (student or tutor) with optional search.

<ParamField query="search" type="String" optional>
  Search term to filter classrooms
</ParamField>

<ParamField query="pagination" type="PaginationParams" default="{}">
  Pagination parameters
</ParamField>

<ResponseField name="classrooms" type="List<Classroom>">
  Returns filtered and paginated classroom list
</ResponseField>

```kotlin
// ClassroomResource.kt:62
@Query
fun userClasses(
    @Name("search") search: String? = null,
    @Name("pagination") @DefaultValue("{}") paginationParams: PaginationParams
): List<Classroom> {
    val user = identityUtil.account
    
    if (identityUtil.isTutor()) {
        return classroomService.getClassroomsPaged(
            tutorId = user.id,
            accountId = user.id,
            search = search,
            page = paginationParams.page!!,
            size = paginationParams.size!!
        )
    }
    
    if (identityUtil.isStudent()) {
        return classroomService.getClassroomsPaged(
            studentId = user.id,
            accountId = user.id,
            search = search,
            page = paginationParams.page!!,
            size = paginationParams.size!!
        )
    }
    
    // For impersonated accounts
    val impersonateAccount = identityUtil.impersonateAccount
    return classroomService.getClassroomsPaged(
        studentId = impersonateAccount.id,
        accountId = user.id,
        search = search,
        page = paginationParams.page!!,
        size = paginationParams.size!!
    )
}
```

## Classroom Lessons

### classroomPendingLessons

Retrieve pending lessons for a classroom, sorted by date and time.

<ParamField query="classroomSignature" type="String" required>
  The classroom signature
</ParamField>

<ParamField query="pagination" type="PaginationParams" default="{}">
  Pagination parameters
</ParamField>

<ResponseField name="lessons" type="List<Lesson>">
  Pending lessons sorted by date (ASC) and time (ASC)
</ResponseField>

```kotlin
// ClassroomResource.kt:139
@Query
fun classroomPendingLessons(
    @Name("pagination") @DefaultValue("{}") paginationParams: PaginationParams,
    classroomSignature: String,
): List<Lesson> {
    return lessonService.getLessons(
        args = LessonPredicateParams(
            classroomSignature = classroomSignature,
            status = LessonStatus.PENDING,
        ),
        pageParams = PageParams(
            page = paginationParams.page!!,
            size = paginationParams.size!!
        ),
        sortBy = listOf(
            Tuple2.of(Lesson_.LESSON_DATE, "ASC"), 
            Tuple2.of(Lesson_.LESSON_TIME, "ASC")
        )
    )
}
```
